# Submission Checklist

## Rule File (.xml)
- [ ] XML is newly developed and functional in LanguageTool.
- [ ] Exactly 3 spelling-error rules implemented.
- [ ] Exceptions handled for each rule.
- [ ] Rule IDs, messages, and suggestions are clear.
- [ ] Examples in XML updated and no placeholders remain.

## Documentation (.pdf)
- [ ] Names, section, and group number included.
- [ ] 3 rules fully documented with OP/MMP basis.
- [ ] 3 examples per rule included.
- [ ] 2 websites/platforms per rule included.
- [ ] LanguageTool version indicated.
- [ ] MS Word plugin screenshot included if applying for bonus.
- [ ] Contribution table included.
- [ ] AI usage declaration included per member.

## Video Demo
- [ ] Rules demonstrated in standalone LanguageTool.
- [ ] MS Word integration shown if applicable.
- [ ] Exceptions explained.
- [ ] All members appear and actively participate.
- [ ] Delivery is spontaneous.

## Final quality checks
- [ ] No false positives in prepared validation set.
- [ ] URLs are accessible and date-accessed is recorded.
- [ ] File names are consistent and clean.
- [ ] Deadline check: March 31, 11:59 PM.
