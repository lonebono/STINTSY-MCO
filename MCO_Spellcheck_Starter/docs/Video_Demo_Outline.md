# Video Demo Outline (Spontaneous Flow)

## Target length
- 4 to 8 minutes

## Segment 1: Intro (30-45 seconds)
- Introduce members, section, and project goal.
- State that rules are based on OP/MMP.

## Segment 2: Rule 1 walkthrough (60-90 seconds)
- Explain formal pattern in plain language.
- Show at least 1 incorrect example flagged and corrected.
- Show one exception case that is not flagged.

## Segment 3: Rule 2 walkthrough (60-90 seconds)
- Explain pattern and rationale from references.
- Run sample text and show output.
- Show exception handling.

## Segment 4: Rule 3 walkthrough (60-90 seconds)
- Explain pattern and rationale from references.
- Run sample text and show output.
- Show exception handling.

## Segment 5: Precision check (45-60 seconds)
- Show a short clean text where no false positives appear.
- Mention test strategy used to verify precision.

## Segment 6: Tooling and closing (45-60 seconds)
- Show LanguageTool version used.
- Show MS Word plugin if integrated.
- Summarize contributions of each member.

## Delivery requirements reminder
- All members appear on camera.
- Presentation should sound natural and not read verbatim.
